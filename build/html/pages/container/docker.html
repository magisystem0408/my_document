例の「鯨」の飼い方(Docker)

dockerコマンド

ビルド

docker image build -t sample:latest .

コンテナの停止

docker container stop sample

コンテナの削除

docker container rm sample

コンテナの確認

docker container ls -a

docker ps

docker ps -a

コンテナのログ確認

docker container logs sample

コンテナの一括削除

docker system prune -a

docker-composeコマンド

ビルド

docker-compose build

コンテナの作成と起動

docker-compose up -d

-dでバックグランドで起動する

コンテナの停止・削除

docker-compose down

コンテナの一覧を表示

docker-compose ps

ログ表示

docker-compose logs

起動中のコンテナにコマンド実行

docker-compose exec <サービス> <コマンド>
